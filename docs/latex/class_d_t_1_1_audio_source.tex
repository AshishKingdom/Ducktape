\doxysubsection{Detailed Description}
Component for playing Audio. 

Attaching this component to an entity lets you play audio. Use the \mbox{\hyperlink{class_d_t_1_1_audio_source_ac0fe6aeae0ea72fd7f19761f9efe5375}{Audio\+Source\+::load\+Sound()}} / \mbox{\hyperlink{class_d_t_1_1_audio_source_abf4e26963822d4bb13e98c5662d35f3e}{Audio\+Source\+::load\+Music()}} methods to load sound/music from a specified path, and the \mbox{\hyperlink{class_d_t_1_1_audio_source_a70c1208a517869e0cdcff7eaed1f7f12}{Audio\+Source\+::play()}}, \mbox{\hyperlink{class_d_t_1_1_audio_source_a320ea7238e35e42982513e0e9b263a53}{Audio\+Source\+::stop()}}, \mbox{\hyperlink{class_d_t_1_1_audio_source_ac6e335c512ea35c5c5f1a8cbf0acfcba}{Audio\+Source\+::pause()}} functions to control the playback of the audio. Ducktape supports the .wav, .ogg, .flac audio formats. Due to licensing issues, .mp3 is not supported as of now.

Usage can be found in the Manual\+: \href{https://ducktapeengine.github.io/manual/audio.html}{\texttt{ https\+://ducktapeengine.\+github.\+io/manual/audio.\+html}}

{\footnotesize\ttfamily void DT\+::\+Audio\+Source\+::load\+Music (\begin{DoxyParamCaption}\item[{std\+::string}]{load\+Path }\end{DoxyParamCaption})}



Play a music from a specified path. 

Use this method to load and play a music from a specified path. Unlike \mbox{\hyperlink{class_d_t_1_1_audio_source_ac0fe6aeae0ea72fd7f19761f9efe5375}{Audio\+Source\+::load\+Sound()}} It doesn\textquotesingle{}t load all the audio data into memory, instead it streams it on the fly from the source file. It is typically used to play compressed music that lasts several minutes, and would otherwise take many seconds to load and eat hundreds of MB in memory.
